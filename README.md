# Showboard Releases

Public download repository for [Showboard](https://showboard.app) — the audiophile music player built for live music collectors.

## Download

**[Download the latest release](https://github.com/maugyi-aces/showboard-releases/releases/latest)**

## Requirements

- macOS 14 (Sonoma) or later
- Apple Silicon or Intel Mac

## What is Showboard?

Showboard is a native macOS music player that unifies your local library with nugs.net, TIDAL, Qobuz, Archive.org, and Phish.in in one audiophile-grade application. Bit-perfect playback, live music intelligence, and a concert diary that connects your real-world show experiences to recordings.
